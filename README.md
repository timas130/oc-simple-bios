# Simple BIOS
A simple EEPROM BIOS for [OpenComputers](https://ocdoc.cil.li/).

## Installing
```bash
wget -f https://raw.githubusercontent.com/timas130/oc-simple-bios/master/code.min.lua && echo "Flashing..." && flash -q code.min.lua "Simple BIOS" && echo "Done!"
```
